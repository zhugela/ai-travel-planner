#### 云南大理 4 天怎么玩?

1. D1 大理古城:五华楼 + 人民路,傍晚看日落
2. D2 环洱海:租电动车,走双廊、喜洲、蝴蝶泉
3. D3 苍山:洗马潭索道,海拔 4000 米,带外套
4. D4 沙溪古镇:安静原生态,适合发呆
预算 3000 元/人(含机票)

#### 西藏 7 天预算多少?

1. 进出藏机票 1500~2500 元
2. 拉萨市内 3 天 800 元(布达拉宫 + 大昭寺 + 八廓街)
3. 纳木错 1 日游 400 元
4. 林芝 3 天 1500 元(含桃花节加价)
总计 4500~5500 元/人,高反药另算

#### 高原反应怎么处理?

1. 提前 1 周服红景天
2. 抵达后多休息、多喝水、忌饮酒
3. 头痛 / 呕吐立即吸氧
4. 严重高反立即下撤到低海拔(昆明 / 林芝)
常用药:葡萄糖 + 红景天 + 头痛粉
