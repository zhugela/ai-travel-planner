#### 第一次出境推荐哪里?

1. 泰国曼谷 + 普吉:免签,中文服务多,机票 1500 元
2. 新加坡:安全、干净、华人多,4 天够
3. 日本东京 + 京都:体验好,需签证,5~7 天

---

#### 申根签证提前多久办?

1. 出发前 3 个月开始准备
2. 预约递交材料 + 录指纹
3. 旺季审理 15~20 工作日
4. 常见材料:护照、身份证、在职证明、银行流水、机票预订单、酒店预订单
注意:首次申根必须去面签的国家

---

#### 出境游保险买哪种?

1. 必含 4 项:医疗救援、紧急转运、个人责任、行李延误
2. 推荐品牌:美亚、安联、平安
3. 申根签证强制 30 万医疗保额
4. 价格参考:7 天 50 万保额约 80~150 元

---

#### 国际航班座位怎么选?

1. 经济舱:前排靠过道,出入方便
2. 长途选紧急出口排,腿部空间大
3. 看风景:南半球左侧、北半球右侧
4. 婴儿摇篮位提前向航司申请

---

#### 换多少外币合适?

1. 国内换 500~1000 美元等值小额现金(应急)
2. 大额用双币信用卡(VISA / Mastercard)
3. 当地 ATM 取现:认银联标志,手续费低
4. 支付宝 / 微信:东南亚 + 日本部分商户支持

---

#### 出境游电话卡怎么解决?

1. 移动 / 联通国际漫游:贵,适合 1~3 天
2. 提前买目的地电话卡:淘宝 / 京东,7 天 30~80 元
3. 当地 eSIM:支持的国家越来越多,iPhone 友好
4. 租随身 WiFi:适合 2~3 人合用,30 元/天
